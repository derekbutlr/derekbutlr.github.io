Describe that the DOM is and an example of how you might interact with it.

The glue that ties everything together is called the document object model (DOM). Your DOM is an object representation of your entire workspace in your document. You can view the DOM through your browser & it represents all of your code in object form. Everything contained within the DOM is called a node- summoned by your browser.
Every HTML Element that you use has a javascript representation that backs it. By having an objrep you can not only access these elements in the DOM, but you can modify and create elements and add them while you work. This could be especially helpful for seeing your live changes as you work without having to switch windows all the time.

Explain the difference between accessing data from arrays and objects.

The value of an array is that you can use them to store a collection of data; an example of some of the types of data you could store in your array are:
- Booleans (True / False)
- Integers (A whole number not surrounded by quotes)
- Strings (something surrounded by single or double quotes)

Arrays are defined in square brackets, and objects can be accessed with parenthesis or dot notation If you have 1000 different entries of wind pressure from a wPS, they would go in an array. Once they are in the array you can iterate over the array and do something on each element, and the same thing on each other element. However conversely, each object is representitive of one whole thing. Say for instance I as a person am an object, the data stored inside my object properties could be my eyes, height, age, etc. But they are all different sets of data so we may want to use them in lots of different ways.

Explain what functions are and why they are useful.

Functions are basically reusable pieces of code (containing a set of statements) that have conditional go-buttons for when they are required. There are lots of built in functions (we learnt some, for instance console.log, alert) but we can also create them for just about anything, and then called upon when needed. For instance:

function toCelsius(fahrenheit) {
return (5/9) * (fahrenheit-32); }

Is a function that converts Fahrenheight to Celcius, however you can also use functions as variable values:

var x = toCelsius(77);
var text = "The temperature is " + x + " Celsius";

This saves us a lot of time in the long run, not having to repeatedly type the same things over and over again and allows us to create much more useful applications with our code.
